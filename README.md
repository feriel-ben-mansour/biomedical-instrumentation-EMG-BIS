# Biomedical Instrumentation – EMG/BIS Signal Acquisition System

**End Of Studies Internship Project** 

## Project Overview
Development of a **wearable EMG and BIS signal acquisition system** with low-power electronics, PCB design, and algorithms for data acquisition and feature extraction.

---

## Key Contributions
- Designed a unique EMG/BIS signal acquisition architecture
- Selected low-power components through analog circuit simulations
- Developed PCB design and prototyping for reliable biological signal acquisition
- Created algorithms for data acquisition, filtering, and feature extraction

---

## Tools & Technologies
Altium Designer - LTSpice - TINA-TI - MATLAB - STM32CubeMX - STM32CubeIDE

---

## Repository Structure
- `PCB-design/` → Altium designer PCB schematics and layouts  
- `Circuit Simulations/` → Analog simulations and component selection  
- `Firmware/` → Embedded code for STM32 microcontrollers  
- `Docs/` → Project documentation and diagrams
